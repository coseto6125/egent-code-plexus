# @egent-code-plexus/__PLATFORM__

Prebuilt `ecp` binary for `__PLATFORM__`.

Do not install this package directly. It is an `optionalDependency` of
[`egent-code-plexus`](https://www.npmjs.com/package/egent-code-plexus); npm
selects the matching platform package automatically via the `os`/`cpu` fields.

```sh
npx egent-code-plexus --help
# or
npm install -g egent-code-plexus
```
