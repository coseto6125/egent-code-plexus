Run MCP server (serve) or list exposed tools (tools)

Usage: gnx admin mcp [OPTIONS] <COMMAND>

Commands:
  serve  Run stdio JSON-RPC MCP server
  tools  List tools that would be exposed by `serve`
  help   Print this message or the help of the given subcommand(s)

Options:
      --graph <GRAPH>  Path to the graph.bin file [default: .gnx/graph.bin]
  -h, --help           Print help
