# cgn group

Manage and query repo groups.

## Subcommands
- `sync <name>`: Build cross-links and extract contracts for the group.
- `status <name>`: Check staleness of group members.
- `contracts <name>`: Inspect the contract registry.
- `impact <name>`: Cross-repo blast radius.
- `find <name>`: Search across all group members.

## Usage
```bash
cgn group <subcmd> <group_name> [args]
```
