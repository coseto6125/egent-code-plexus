Manage repo group membership

Usage: ecp admin group [OPTIONS] <COMMAND>

Commands:
  add     Add a repo to a group (auto-creates group)
  remove  Remove a repo from a group (auto-deletes empty group)
  help    Print this message or the help of the given subcommand(s)

Options:
      --graph <GRAPH>  Path to the graph.bin file [default: .ecp/graph.bin]
  -h, --help           Print help
